## save 
可以保存**layout**布局和**setting**参数
布局就是每次打开的时候窗口出现的位置和大小；参数就是一些初始参数，有些参数很重要一定要填，所以最好保存一个初始状态这样每次加载就可以确保不会漏。

## load
可以进行**layout**布局和**setting**参数的加载。

## Quit
点击quit之后等一段时间让程序正常退出。
